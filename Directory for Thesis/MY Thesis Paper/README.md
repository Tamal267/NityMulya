# Thesis Paper - AI-Enhanced Complaint Management System

## Files

- **thesis_paper.tex** - LaTeX source file (enhanced version)
- **thesis_paper.pdf** - Generated PDF document (19 pages)
- **generate_pdf.py** - Python script to compile LaTeX to PDF
- **ENHANCEMENT_SUMMARY.md** - Detailed summary of all improvements made

## Quick Start

### Generate PDF from LaTeX

Simply run the Python script:

```bash
python3 generate_pdf.py
```

Or if you have execute permissions:

```bash
./generate_pdf.py
```

### Manual PDF Generation

If you prefer to use LaTeX commands directly:

```bash
pdflatex thesis_paper.tex
pdflatex thesis_paper.tex  # Run twice for references
```

## Requirements

### For PDF Generation
- Python 3.x
- LaTeX distribution (texlive-latex-base or equivalent)

### Install LaTeX on Linux

**Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install texlive-latex-base texlive-latex-extra
```

**Fedora/RHEL:**
```bash
sudo dnf install texlive-scheme-basic
```

**Arch Linux:**
```bash
sudo pacman -S texlive-core
```
